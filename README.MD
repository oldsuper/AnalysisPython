###大盘指数
关注内容：尾盘的量与未来1、3天走势关系，与第二天高低开关系，  
扫描尾盘10分钟、15分钟、30分钟（两点半后跳水、大涨）  
大盘日kdj低位金叉、高位死叉与未来5天走势——中短期效应  
大盘周kdj低位金叉、高位死叉与未来20天走势——中期效应  
MACD顶背离、底背离提醒

###关注股

关注内容：量，与大盘的差异性
kdj低位金叉
